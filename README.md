# Project Title

League Guide: A Beginner's Guide to League of Legends

## Project Description

League Guide is a responsive, single-page website created to help new players learn the basics of League of Legends. The page introduces the game's objective, explains the five player roles, provides beginner tips, and recommends champions that are easy to learn. The website was built using HTML and CSS as part of a digital media course project.

## Intended User or Player

This interface is designed for people who are new to League of Legends and want a simple introduction before playing the game.

## User / Player Goal

The goal is to help users understand the basics of League of Legends, learn the responsibilities of each role, discover helpful beginner tips, and become familiar with a few easy-to-play champions.

## Design Choices

**Layout:**
- Single-page layout with navigation links that smoothly scroll to each section.
- Information is separated into clear sections using cards and images.

**Color:**
- Dark blue and gold were used to match the League of Legends theme.
- White text provides contrast and improves readability.

**Typography:**
- Arial was used because it is simple, clean, and easy to read.

**Visual Hierarchy:**
- The hero section is the first thing users see.
- Large headings separate each section.
- Cards organize information into smaller, readable pieces.

**Feedback States:**
- Navigation buttons and cards include hover effects.
- Smooth scrolling helps users navigate between sections.

**Responsive Design:**
- On smaller screens, the layout stacks vertically.
- Navigation wraps onto multiple lines, and cards stack to improve readability on mobile devices.
